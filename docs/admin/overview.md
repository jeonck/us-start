# 📋 행정 가이드 (Administration)

미국 정착에 필요한 주요 행정 절차 안내입니다.

## 1. SSN (사회보장번호) 발급
SSN은 미국 생활의 '주민등록번호'와 같습니다.
- **신청 시기**: 입국 후 약 2주 뒤 (데이터 업데이트 시간 필요)
- **장소**: 가까운 Social Security Administration Office 방문
- **필수 서류**: 여권, I-94(입국 증명), 비자 관련 서류

## 2. 텍사스 운전면허증
미국에서는 운전면허증이 가장 보편적인 신분증(ID)입니다.
- **절차**: ITD(Impact Texas Drivers) 교육 이수 -> 필기 시험 -> 실기 시험
- **준비물**: 거주 증명 서류 2종(공과금 고지서, 집 계약서 등), SSN, 여권/비자

## 3. 차량 등록 (Vehicle Registration)
- **타주/신규 차량**: 텍사스 주 입국 후 30일 이내에 등록해야 합니다.
- **필수 사항**: 자동차 보험 가입, 차량 검사(Inspection) 통과
