Berkeley와 Stanford 연구진이 제안한 TAG(Table-Augmented Generation) 기술을 소개합니다. 기존 Text2SQL과 RAG의 한계를 극복해 구조화된 데이터와 LLM을 결합하고, movies.csv 실습 예제를 통해 HNSW 기반 시맨틱 검색 대비 적은 토큰으로 고품질 응답을 생성하는 방법을 비교 검증합니다.

{/* truncate */}

## 핵심 내용

- **TAG란**: 구조화된 테이블 데이터 + LLM을 결합해 복잡한 자연어 질문에 정교한 답변 제공
- **기존 한계 극복**: Text2SQL(SQL 변환 오류)과 RAG(비정형 데이터 중심)의 단점 보완
- **실습**: movies.csv로 TAG 구현 및 HNSW 시맨틱 검색과 성능 비교
- **결과**: 더 적은 토큰으로 고품질 응답 생성
- **활용**: 기업 내 구조화 데이터 기반 자연어 질의응답 서비스 구축

## 주요 기술 키워드

`TAG` `Table-Augmented Generation` `RAG` `Text2SQL` `LLM` `NLQ` `HNSW` `Structured Data` `Semantic Search`
