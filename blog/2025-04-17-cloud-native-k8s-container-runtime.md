Docker에서 containerd로의 컨테이너 런타임 전환 역사를 추적하며, OCI·CRI 표준 기반의 경량화된 아키텍처 덕분에 Kubernetes 설치가 얼마나 단순해졌는지를 설명합니다. 온프레미스 환경에서의 12단계 실무 구축 가이드를 제공합니다.

{/* truncate */}

## 핵심 내용

- **런타임 역사**: Docker → containerd로의 산업 전환 과정과 배경
- **표준화**: OCI(Open Container Initiative)와 CRI(Container Runtime Interface) 준수의 중요성
- **경량화**: containerd의 단순화된 아키텍처로 K8s 설치 복잡도 감소
- **실무 가이드**: 온프레미스 환경 K8s 구축 12단계 절차
- **활용 환경**: 클라우드 없이 사내 서버에서 쿠버네티스 운영

## 주요 기술 키워드

`Kubernetes` `containerd` `Docker` `OCI` `CRI` `Cloud-Native` `On-premise` `Container Runtime`
