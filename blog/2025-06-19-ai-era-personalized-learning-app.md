에빙하우스 망각곡선 이론을 적용한 GRE 학습 앱 'GRE Mind'를 LLM과 MCP 파일시스템을 활용해 몇 분 만에 프로토타입으로 완성한 사례입니다. 개발자의 역할이 반복적 구현에서 의사결정 중심으로 전환되는 새로운 개발 패러다임을 제시합니다.

{/* truncate */}

## 핵심 내용

- **기획**: LLM으로 에빙하우스 망각곡선 기반 학습 앱 아이디어를 빠르게 구체화
- **개발**: 프롬프트 기반 개발 + MCP 파일시스템으로 HTML/JS/CSS 프로토타입 즉시 생성
- **기술 스택**: Claude, MCP, SQLite, JSON, HTML/JavaScript/CSS
- **인사이트**: 반복 구현 → 의사결정 중심으로 개발자 역할 전환
- **결과**: 아이디어에서 작동하는 앱까지 수분 내 완성

## 주요 기술 키워드

`LLM` `Claude` `MCP` `Prototype` `HTML` `JavaScript` `SQLite` `Forgetting Curve` `Low-Code`
