Spring Boot로 간단한 애플리케이션을 처음부터 만들어보는 실무 가이드입니다. Repository 생성부터 서버 기동, 빌드 환경 구성, GitHub 코드 관리까지 단계별로 따라할 수 있으며, 커밋 이력을 통해 코딩 단계별 진행 상황도 확인할 수 있습니다.

{/* truncate */}

## 핵심 내용

- **환경 구성**: JDK, Gradle, Spring Boot 프로젝트 초기 설정
- **핵심 기술**: JPA + H2 Database + Mustache 템플릿 엔진
- **REST API**: Lombok을 활용한 간결한 컨트롤러·서비스 구현
- **코드 관리**: GitHub 커밋 기반 단계별 이력 관리
- **빌드·기동**: Gradle 빌드 및 Spring Boot 서버 실행

## 주요 기술 키워드

`Spring Boot` `JDK` `Gradle` `JPA` `H2 Database` `Mustache` `Lombok` `REST API` `GitHub`
